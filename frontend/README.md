# Frontend - Product Importer UI

This folder will contain the React + Vite frontend application.

## Setup

```bash
npm create vite@latest . -- --template react
npm install
npm run dev
```

## Features to implement:
- Product list with filtering and pagination
- CSV upload with progress bar
- Product CRUD (create, read, update, delete)
- Bulk delete with confirmation
- Webhook management
- Real-time progress updates via SSE

See main `README.md` for full setup instructions.
